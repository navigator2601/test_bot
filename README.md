# Telegram Бот Помічник-Монтажник

Цей проект створений для допомоги монтажникам кондиціонерів шляхом надання доступу до бази даних, нормативної документації, інтерактивного тестування та інших корисних функцій.

## Структура проекту

```
project/
├── main.py                    # Головний файл запуску бота
├── config.py                  # Файл конфігурації
├── handlers/                  # Обробники команд
│   ├── __init__.py
│   ├── start_handler.py
│   ├── menu_handler.py
│   ├── test_handler.py
│   └── ...
├── keyboards/                 # Клавіатури
│   ├── __init__.py
│   ├── main_menu.py
│   └── test_menu.py
├── database/                  # Модуль роботи з базою даних
│   ├── __init__.py
│   ├── connection.py
│   ├── queries.py
│   └── models.py
├── ai/                        # Інтеграція ШІ
│   ├── __init__.py
│   └── chat_gpt.py
├── utils/                     # Утиліти
│   ├── __init__.py
│   └── helpers.py
├── static/                    # Статичні файли
│   ├── normative_docs/        # Нормативна база
│   │   └── some_doc.pdf
│   └── ...
├── tests/                     # Тести
│   ├── __init__.py
│   ├── test_handlers.py
│   ├── test_database.py
│   └── ...
├── requirements.txt           # Список залежностей
├── .env                       # Секретні дані
├── .gitignore                 # Ігнорування файлів git
└── README.md                  # Документація проекту
```

## Запуск проекту

1. Встановіть залежності:
   ```
   pip install -r requirements.txt
   ```

2. Створіть файл `.env` і додайте секретні змінні:
   ```
   BOT_TOKEN=ваш_токен_бота
   DB_URL=ваш_рядок_підключення_до_бази_даних
   ```

3. Запустіть бот:
   ```
   python main.py
   ```

## Функціонал
- **Інформація через меню**: Користувач може вибрати бажану функцію через клавіатуру.
- **Робота з базою даних**: Використовується PostgreSQL для зберігання та отримання даних.
- **Нормативні документи**: Доступ до статичних файлів із нормативною інформацією.
- **Інтерактивне тестування**: Користувач може проходити тести.
- **Інтеграція ШІ**: У майбутньому буде додано функціонал GPT для розумних відповідей.
