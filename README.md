# Telegram Bot "Test Bot"

Простий Telegram-бот, створений на Python. 

## 📋 Особливості
- Відповідає на команду `/start` із привітанням.
- Легко розширюється для нових функцій.

## 🛠️ Як запустити

1. Клонувати репозиторій:
   ```bash
   git clone https://github.com/navigator2601/test_bot.git
   cd test_bot
   ```

2. Встановити залежності:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   pip install -r requirements.txt
   ```

3. Налаштувати файл `.env`:
   ```
   TOKEN=ваш_токен_бота
   ```

4. Запустити бота:
   ```bash
   python main.py
   ```

## 📞 Питання чи пропозиції
Звертайтеся через Telegram або відкрийте issue в репозиторії.
